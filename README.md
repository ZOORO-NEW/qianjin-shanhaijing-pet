# qianjin-shanhaijing-pet · 神兽宠物提示词生成器

输入一只神兽名称（山海经 / 搜神记 / 淮南子 / 神异经 / 庄子 / 列子等古籍），自动生成 **3 个不同风格版本** 的 AI 绘画提示词：

| 风格 | 气质 | 适用 |
|------|------|------|
| 🏮 国风水墨版 | 回归原典，古籍插图 / 宋元工笔 | 书封、卷轴、典籍风插画 |
| 🧸 Q版萌宠版 | 大头小身、软萌陪伴 | 桌面陪伴、盲盒、贴纸 |
| 🌑 暗黑史诗版 | 诡谲威严、电影感 | 游戏原画、收藏级立绘 |

每版输出：**英文提示词（可直接喂 MJ/SD）** + **简体中文设计说明** + **推荐参数**。

## 快速开始

告诉 WorkBuddy（已安装本技能）：

> 生成「九尾狐」的神兽提示词

即返回九尾狐的 国风 / 萌系 / 暗黑 三版提示词。

也支持别名与诸子古籍神兽，例如：`烛龙`、`鲲鹏`、`白泽`、`穷奇`、`青龙`、`梼杌` 等。

## 输出语言规则

- 提示词主体：**英文**（喂绘图模型）
- 设计说明：**简体中文**，**不使用繁体**

## 风格规则来源

风格语言继承自 `qianjin-ip-design`（6 大风格 × 8 维设计体系），此处将「IP 角色」替换为「古籍神兽」，并按宠物养成场景适配。
